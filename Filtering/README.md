# ER-Morphology

## How to run our code
1. Install necessary dependencies and libraries using the requirements.txt
2. Download TIF images from: [SFU VAULT](https://vault.sfu.ca/index.php/s/GbPAXWICEl4gCwX)
3. Run gui.py from terminal: ` python3.8 gui.py `
4. Select Downloaded TIF Image
5. Allow program time to process ( <1 min )
6. Change method/slice displayed using sliders
